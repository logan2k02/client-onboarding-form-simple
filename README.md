### 🚀 Setup

#### 1. Clone the repo

```bash
git clone https://github.com/logan2k02/client-onboarding-form-simple.git
cd client-onboarding-form-simple
```

#### 2. Install Dependencies

```bash
npm install
```

#### 3. Run Tests

```bash
npx vitest
```
